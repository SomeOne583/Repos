Ajax: Asynchronous JavaScript and XML

API: Conjunto de subrutinas, funciones y procedimientos

body: Etiqueta HTML que define el cuerpo (lo visual) de una página.

button: Etiqueta HTML para crear un botón

Caché: La memoria caché de un procesador, es un tipo de memoria volátil (como la memoria RAM), pero muy rápida.

div: Etiqueta HTML para crear una sección

DOM: Estructura que contiene los elementos de una página.

Embed: Etiqueta HTML que integra una aplicación externa.

Framework: Conjunto estandarizado de conceptos, prácticas y criterios para enfocar un tipo de problemática particular que sirve como referencia.

CGI Computer Generated Imagery: Las imágenes generadas por computadora o por ordenador o CGI del inglés «Computer Generated Imagery (CGI)», son el resultado de la aplicación de la infografía y más específicamente, de los gráficos 3D generados por ordenador, en el arte, los videojuegos, las películas, los programas y anuncios de televisión, las animaciones, los simuladores y la simulación en general y, también, en los medios impresos.

Gopher: Servicio de internet que permite el acceso a la información en forma de árbol

Grid: Sistema bidimensional para dividir un elemento HTML según una cuadrícula.

Hipermedia: Conjunto de métodos o procedimientos para escribir, diseñar o componer contenidos que integren soportes tales como: texto, Imagen, video, audio, mapas y otros soportes de información emergentes

Host: Se usa en informática para referirse a las computadoras u otros dispositivos (tabletas, móviles, portátiles) conectados a una red que proveen y utilizan servicios de ella

HTML: Lenguaje de marcado que permite diseñar páginas web.

img: Etiqueta HTML que permite agregar una imagen.

jQuery: Librería que permite agilizar el código de JavaScript.

json: Es un formato de texto sencillo para el intercambio de datos

LISTSERV: Es el nombre de un software que gestiona todo tipo de listas de correo electrónico, tanto las que únicamente envían algún tipo de información, como aquellas otras que permiten la comunicación en dos sentidos

name: Atributo con el que JavaScript detecta un contenido en un HTML.

NETIQUETTE: net (que se traduce como “red”) y etiquette (que alude a la “etiqueta” con relación al ceremonial). La netiquette se vincula a las pautas de comportamiento en Internet. Se trata, por lo tanto, de las reglas de etiqueta que deben aplicarse en un entorno virtual para mantener vínculos respetuosos y a tono con las normas de convivencia.

Servlet: Los servlets son modulos java que nos sirven para extender las capacidades de los servidores web. son programas para los servidores, mientras que los applets son programas para los clientes y los middlets los programas para microdispositivos.

Tipos de dominio: .com, .net, .org, .gob, etc.: Un dominio es el nombre con el que se conoce a un sitio web. Hay dominios para todo tipo de empresa o circunstancia y es el dominio quien definirá cómo lo encontrarán en Internet.
